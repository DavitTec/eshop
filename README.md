## Install

1. npx create-next-app
   ```bash
   mkdir eshop  && cd eshop
   git init
   cd ..
   npx create-next-app@latest eshop
   cd eshop
   git add .
   git commit -m "First commit - installed NEXT App"
   ```
   
   Now open, eg

   ```bash
    atom .  # for Atom or 'code .' for VSCode

    ```
